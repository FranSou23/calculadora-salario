# Calculadora de salário

## 📝 Descrição
Baseado no valor por hora e na quantidade de horas trabalhadas calcula o quanto o usuario
recebe por mês

## 📚 Funcionalidades
** Salário por mês**: sálario hora * horas trabalhadas
